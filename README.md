# Classroom Doubt Solver (ML-based)

This project is a **real-time classroom doubt solver**.  
Students can type their doubts and immediately get answers without needing a teacher.

### Current Status (10 December)

- ✅ Project repo created
- ✅ Virtual environment & dependencies set up
- ✅ Basic TF-IDF based doubt solver (baseline)

### Tech Stack

- Python
- scikit-learn (TF-IDF & cosine similarity)
- pandas, numpy

More features, UI, and documentation will be added over the next few days.
